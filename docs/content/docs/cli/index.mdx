Penny provides several subcommands for managing your reverse proxy setup.

## Commands

| Command | Description |
|---------|-------------|
| `penny serve` | Start the reverse proxy |
| `penny check` | Validate your configuration |
| `penny systemd` | Manage Penny as a systemd service |
| `penny dokku` | Dokku integration commands |
