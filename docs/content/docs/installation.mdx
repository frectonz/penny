## Install via Shell Script

The easiest way to install Penny is using the install script:

```sh
curl -LsSf https://pennyproxy.com/install.sh | sh
```

## Download Prebuilt Binaries

You can also download prebuilt binaries from the [GitHub releases page](https://github.com/frectonz/penny/releases).

## From Source

Requires Rust (2024 edition) and [pnpm](https://pnpm.io/) for the dashboard UI.

```bash
git clone https://github.com/frectonz/penny.git
cd penny

# Build the dashboard UI
cd ui && pnpm install && pnpm build && cd ..

# Build penny
cargo build --release
```

The binary will be at `target/release/penny`.

## Verify Installation

```bash
penny --help
```
